# iac-ct-openspeedtest
Infrastructure as Code for iac-ct-openspeedtest deployment.
## Overview

This repository contains the infrastructure as code for iac-ct-openspeedtest deployment using Docker and Ansible.

## Files

- `deploy-iac-ct-openspeedtest.sh` - Main deployment script
- `configure-iac-ct-openspeedtest.sh` - Configuration script
- `ansible-playbook.yml` - Ansible playbook
- `docker-compose.yml` - Docker compose configuration

## Getting Started

1. Clone this repository
2. Run `./deploy-iac-ct-openspeedtest.sh` to deploy
